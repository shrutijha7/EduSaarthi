Q1: The course 'COMPUTER ARCHITECTURE AND ORGANIZATION' is exclusive to students in the Department of Computer Science and Engineering.
Answer: False
Explanation: The course description explicitly states that the syllabus is '(Common to CSE&IT),' indicating that it is shared between Computer Science and Engineering (CSE) and Information Technology (IT).

Q2: The course content includes instruction on basic technical writing skills.
Answer: True
Explanation: The introductory description of the course notes that 'Basic technical writing skills are also taught in this class.'

Q3: One of the stated course outcomes is the ability to critique the performance issues related to cache memory and virtual memory.
Answer: True
Explanation: Course Outcome 5 states that students will be able to 'Critique the performance issues of cache memory and virtual memory.'

Q4: Topics related to RAID (Redundant Array of Independent Disks) are covered in the unit dedicated to Input/Output (I/O).
Answer: False
Explanation: RAID is listed as a topic in Unit IV, titled 'MEMORY SYSTEM,' alongside Cache Memory and Virtual Memory, not in the dedicated Input Output unit (Unit V).

Q5: The comparison between Complex Instruction Set Computer (CISC) and Reduced Instruction Set Computer (RISC) is covered within the second unit of the syllabus.
Answer: True
Explanation: Unit II, under the Central Processing Unit section, specifically lists 'Complex Instruction Set Computer (CISC) Reduced Instruction Set Computer (RISC), CISC vs RISC' as topics.
